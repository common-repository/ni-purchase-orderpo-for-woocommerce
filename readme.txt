Ni Purchase Order (PO) For WooCommerce gives you the option to create the product purchase order for product vendor or supplier.   

== Description ==
<p>
Ni Purchase Order (PO) For WooCommercegives you the option to create the product purchase order. Setting provides theoption to set a bailing address, shipping address, shop name, shop address, shoplogo and signature in purchase order invoice and option to print or export thepurchase order into PDF.
</p>
<p>
Stock report show the all product list simple and variation product, filter by product name, product sku, mange stock and stock status. Stock report also shows the product stock valuation of each product.
</p>

<strong>Key feature for purchase order system for WooCommerce</strong>
<ul>
	<li>Create purchase order</li>
	<li>Create or select the existing product vendor for purchase order (PO) </li>
	<li>Set the purchase order (PO) status to pending or received</li>
	<li>Create the purchase order (PO) invoice and Print or export to PDF</li>
	<li>Option to set the company logo and signature in purchase invoice</li>
	<li>Option to set shop billing address or shipping address</li>
	<li>Footer notes</li>
	<li>Export to PDF or HTML print </li>
	<li>Stock Report and Stock valuation report</li>
</ul>

== Installation ==

1. Use the WordPress plugin installer or upload files to WordPress plugin directory.
2. Activate "Ni Purchase Order(PO) For WooCommerce" plugin from admin panel.
3. Admin Menu Name "PO".
